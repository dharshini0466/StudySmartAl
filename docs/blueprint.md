# **App Name**: StudySmart AI

## Core Features:

- AI-Powered Content Generation: Generates notes, summaries, MCQs, and flashcards from user-input topics using AI, with the LLM using a tool for reasoning.
- User Authentication: Secure user registration and login to manage personal content.
- Content History: Stores and displays a history of generated learning content, allowing users to revisit past topics and materials.
- Quiz Module: Allows users to take AI-generated quizzes and view their results immediately.
- PDF Download: Enables users to download generated content in PDF format for offline study.
- Topic Input: Allows users to input study topics for content generation.
- MongoDB integration: Storing the topic name, content type and creation date.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire confidence and focus.
- Background color: Light blue (#E8EAF6), a very low saturation variation on the primary that creates a calm learning environment.
- Accent color: Purple (#7E57C2) to highlight important actions.
- Body text: 'Inter', a sans-serif font providing a clean and modern reading experience.
- Headline font: 'Space Grotesk', a sans-serif font providing a computerized, techy feel.
- Use consistent and clear icons to represent different content types (notes, quizzes, etc.).
- Clean, intuitive layout with clear sections for content generation, history, and settings.
